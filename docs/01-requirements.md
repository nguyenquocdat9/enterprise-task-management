# Requirements

## Project Name

Enterprise Task Management

## Goal

Xây dựng hệ thống quản lý công việc theo mô hình công ty.

## Main Features

### Authentication

- User register
- User login
- JWT authentication

### Workspace

- Create workspace
- View workspace list
- Update workspace
- Delete workspace

### Project

- Create project
- View projects
- Update project
- Delete project

### Task

- Create task
- View task list
- Update task
- Change task status
- Assign task to user
- Delete task

### Comment

- Add comment to task
- View comments
- Delete own comment

## User Roles

### ADMIN

- Manage all users
- Manage all workspaces
- Manage all projects

### MANAGER

- Manage projects
- Assign tasks
- Update task status

### MEMBER

- View assigned tasks
- Update own tasks
- Comment on tasks
